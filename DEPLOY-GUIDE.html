SHIELD OS

DEPLOYMENT GUIDE — ALL PLATFORMS — COPY & PASTE READY
⚙️ STEP 0 — PROJECT SETUP (DO THIS FIRST, ONCE)
1 — INSTALL NODE.JS
Download from nodejs.org → Install LTS version. Verify:
node --version
npm --version
2 — CREATE REACT PROJECT
Run this once. Creates a Vite-powered React app in a folder called shield-os:
npm create vite@latest shield-os -- --template react
cd shield-os
npm install
3 — REPLACE THE APP FILE
Delete the default App content and paste your shield-os.jsx content into:
src/App.jsx   ← paste the entire shield-os.jsx content here
Make sure the last line is: export default function ShieldOS() { ... } — it already is if you used the upgraded file.
4 — UPDATE MAIN ENTRY
Open src/main.jsx and replace everything with:
import { StrictMode } from 'react'
import { createRoot } from 'react-dom/client'
import ShieldOS from './App'

createRoot(document.getElementById('root')).render(
  <StrictMode>
    <ShieldOS />
  </StrictMode>
)
5 — TEST LOCALLY
npm run dev
✓ Open http://localhost:5173 — should show Shield OS running perfectly.
6 — PUSH TO GITHUB (needed for Vercel / Cloudflare / Netlify auto-deploy)
git init
git add .
git commit -m "Shield OS v4.0"
# Then create a repo on github.com and run the commands GitHub shows you

▲ VERCEL FREEFASTESTEASIEST
METHOD A — CLI (1 COMMAND)
Install Vercel CLI globally and deploy — that's it:
# Install once globally
npm install -g vercel

# Inside your shield-os folder:
vercel

# Answer the prompts:
# ? Set up and deploy? → Y
# ? Which scope? → your username
# ? Link to existing project? → N
# ? Project name → shield-os
# ? In which directory is your code? → ./
# ? Override settings? → N
# Done! Your live URL appears instantly.
✓ Live URL: https://shield-os-xxxx.vercel.app — Done in ~20 seconds.
METHOD B — GITHUB AUTO-DEPLOY
Connect once, every git push auto-deploys.
1. Go to: vercel.com → Sign up with GitHub (free)
2. Click "Add New Project"
3. Import your shield-os GitHub repo
4. Settings:
   Framework Preset: Vite
   Build Command:    npm run build
   Output Directory: dist
5. Click "Deploy"
✓ Every git push = instant new deployment automatically.
☁ CLOUDFLARE PAGES FREEBEST CDN
METHOD A — GITHUB CONNECT
1. Go to: pages.cloudflare.com → Sign up (free)
2. Click "Create a project" → "Connect to Git"
3. Authorize GitHub → Select your shield-os repo
4. Configure build:
   Project name:     shield-os
   Production branch: main
   Framework preset:  Vite
   Build command:    npm run build
   Build output dir: dist
5. Click "Save and Deploy"
✓ Live at: shield-os.pages.dev — Unlimited requests, global CDN.
METHOD B — WRANGLER CLI
# Build first
npm run build

# Install Cloudflare CLI
npm install -g wrangler

# Login to Cloudflare
wrangler login

# Deploy the dist folder directly
wrangler pages deploy dist --project-name=shield-os
Repeat the last command every time you want to redeploy.

REQUIRED — CREATE netlify.toml (for React Router)
Create this file in your project root — prevents 404 on page refresh:
[[redirects]]
  from = "/*"
  to = "/index.html"
  status = 200
Save as netlify.toml in the root of your project (same level as package.json).
⬡ GITHUB PAGES FREE
STEP 1 — UPDATE vite.config.js
Open vite.config.js and set the base to your repo name:
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()],
  base: '/shield-os/',   // ← your GitHub repo name here
})
STEP 2 — INSTALL gh-pages PACKAGE
npm install --save-dev gh-pages
STEP 3 — ADD DEPLOY SCRIPTS TO package.json
Open package.json and add these two lines inside "scripts":
"scripts": {
  "dev": "vite",
  "build": "vite build",
  "preview": "vite preview",
  "predeploy": "npm run build",     ← ADD THIS
  "deploy": "gh-pages -d dist"      ← ADD THIS
}
STEP 4 — DEPLOY
npm run deploy
✓ Live at: https://YOUR-USERNAME.github.io/shield-os/
STEP 5 — ENABLE IN GITHUB SETTINGS
1. Go to your repo on github.com
2. Settings → Pages
3. Branch: gh-pages → /(root)
4. Save → Wait 1-2 minutes
🌐 CUSTOM DOMAIN (ALL PLATFORMS)
Want shieldos.app instead of shield-os.vercel.app? Buy a domain for ~$10/year on Namecheap or Cloudflare Registrar, then: 
Vercel:          Project → Settings → Domains → Add domain
Cloudflare Pages: Project → Custom domains → Add
Netlify:         Site → Domain management → Add custom domain
GitHub Pages:    Settings → Pages → Custom domain → Enter domain

# Then add this DNS record at your domain registrar:
Type: CNAME
Name: @ (or www)
Value: [the URL they give you, e.g. shield-os.vercel.app]
✓ SSL/HTTPS is automatic and free on all these platforms.
